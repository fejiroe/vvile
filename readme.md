simple modal editor in rust.

'i', 'a', 'x', 's', 'r' from vi mostly work as expected
'ctrl+q' to exit, 'ctrl+s' to save the file in normal mode

"cargo build --release" and copy wherever appropriate or "cargo install --path ."
